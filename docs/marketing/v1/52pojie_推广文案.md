## 🚀 使用方法

### 在线使用（推荐）
1. 打开 https://pipeak.share4y.cn/
2. 可以直接体验离线练习
3. 注册后享受完整功能

### 本地部署
```bash
# 下载源码
git clone https://github.com/prehisle/pipeak.git

# 安装依赖并启动
cd pipeak/backend
pip install -r requirements.txt
python run.py

cd ../frontend  
npm install
npm run dev
```
