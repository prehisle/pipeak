# 你家小孩真的喜欢数学吗？试试这个LaTeX速成训练器就知道了

## 一个简单的测试

给孩子（或你自己）10分钟，看能不能写出这个公式：

```
∫₀^∞ e^(-x²) dx = √π/2
```

写不出来？别担心，大多数人都写不出来。

但如果用LaTeX，只需要这样：
```latex
\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
```

渲染效果：
$$\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$
