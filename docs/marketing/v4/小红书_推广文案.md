## 做个小测试

让孩子试试写出这个公式：

```
圆的面积 = π × 半径²
```

用键盘能打出来吗？大多数打不出来。

但如果学会LaTeX，只需要：
```latex
S = \pi r^2
```

渲染效果：
$$S = \pi r^2$$

简洁、美观、专业。
